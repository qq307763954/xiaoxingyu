小星语 - 原型设计展示

针对3-6岁儿童的智能AI手环管理应用

用户体验分析

目标用户

主要用户：3-6岁儿童的家长

  • 关注儿童语言发展的家长
  • 希望通过科技手段辅助儿童语言学习
  • 需要专业指导以促进儿童语言能力
  • 期望有记录和跟踪功能以观察孩子进步

次要用户：儿童语言发展专家

  • 为家长提供专业指导的语言发展专家
  • 需要了解儿童语言数据的教育工作者
  • 希望通过远程方式提供咨询服务的专家

核心需求

智能监测

通过AI手环实时监测和记录儿童语言表达和情绪状态，为家长提供客观数据。

专业指导

提供科学的语言训练计划和专家咨询，帮助家长科学引导儿童语言发展。

记录与分析

全面记录儿童语言互动，分析语言发展情况，展示进步轨迹和发展趋势。

交互逻辑

产品的核心交互逻辑围绕"监测-记录-分析-指导"四个环节展开：

  1. 数据采集：通过AI手环收集儿童语言数据和情绪状态
  2. 智能分析：对采集的数据进行分析，识别语言发展情况
  3. 记录展示：将分析结果以直观方式呈现给家长
  4. 专业指导：基于分析结果提供个性化训练计划和专家建议
  5. 执行反馈：记录训练效果，形成闭环，持续优化

产品界面规划

信息架构

产品采用四大核心模块结构，通过底部导航栏快速切换：

  1. 首页：汇总核心信息，展示孩子状态、训练计划和数据概览
  2. 记录：详细记录与孩子的互动历史，便于回顾和分析
  3. 专家：专业咨询入口，连接儿童语言发展专家
  4. 我的：个人中心，管理设置、订阅和账户信息

关键页面功能规划

首页

  • 设备状态卡片：显示连接状态和电量
  • 情绪状态监测：实时展示孩子的情绪变化
  • 今日训练：针对性语言训练计划
  • 数据概览：语言发展趋势图表

记录页

  • 设备选择器：切换不同已连接设备
  • 时间筛选器：按日期查看历史记录
  • 分类标签：按类型筛选互动内容
  • 详细记录：查看完整对话内容

专家页

  • 紧急咨询：快速连接在线专家
  • 专家筛选：按专业领域选择专家
  • 专家详情：查看资质和评价
  • 预约/咨询：文字/语音/视频咨询

我的页面

  • 用户信息：账户和会员状态
  • 快速入口：核心功能快捷方式
  • 会员管理：订阅和特权查看
  • 设置与支持：应用设置和帮助

主导航页面

首页

首页展示儿童实时情绪状态、训练计划进度和语言发展概览。

设计理念：
